## 特性

- 多通道 AI 助手能力（网关统一接入）
- 支持本地化部署与自托管运行
- 支持桌面端、移动端与浏览器扩展协同
- 提供可扩展的插件与自动化能力

## 安装方法

### 方式一：全局安装（推荐）

```bash
npm install -g pansclaw@latest
# 或
pnpm add -g pansclaw@latest
```

### 方式二：初始化与引导

```bash
pansclaw onboard --install-daemon
```

### 方式三：从源码安装

```bash
git clone https://github.com/dashiming/pansclaw.git
cd pansclaw
pnpm install
pnpm build
```
