PansClaw Browser Relay

Click the toolbar button on a tab to attach / detach.

First-time setup

Set the relay port to 18792 and paste your Gateway token once. The extension will remember it.

Quick setup

  1. Keep the relay port at 18792.
  2. Paste your gateway.auth.token into the Gateway token field below.
  3. Click Save & Open Dashboard. On success, the Dashboard opens automatically.

If you see a red ! badge on the extension icon, open this page again. It will tell you whether the problem is a missing token, a wrong port, or a relay that is not running.

Full guide (install, remote Gateway, security): Documentation

Relay connection

Default port: 18792. Extension connects to: http://127.0.0.1:<port>/. Gateway token must match gateway.auth.token.
How to get the token

Open your PansClaw config file and copy gateway.auth.token.
